# Theme images

Static images for the AI Awareness Day theme. Use `aiad_theme_image_uri( 'logos/example.png' )` in PHP — do not reference a `public/` folder at the theme root.

| Directory | Contents |
|-----------|----------|
| `favicon/` | Site favicon fallback (see also Site Identity) |
| `logos/` | Partner / backer logo files for reference or imports |
| `polygon-shapes/` | Principle theme polygon SVGs (`safe-polygon.svg`, etc.) |
| `misc/` | Other static artwork |
