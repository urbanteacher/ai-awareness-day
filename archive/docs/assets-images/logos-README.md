# Partner / backer logos

Reference copies for imports or customizer setup. Live partner cards use the **Partner** CPT featured images or customizer URLs — not these paths automatically.

**URL in PHP:** `aiad_theme_image_uri( 'logos/cas.png' )`

## Guidelines

- **Format:** PNG, JPG, WebP, or SVG
- **Size:** Under 100KB each when possible
- **Background:** Transparent or white preferred
